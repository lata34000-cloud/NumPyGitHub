# NumPy Basics

## What is NumPy?
NumPy is a Python library used for working with numerical data.  
It is mainly used for arrays, mathematical operations, and scientific computing.

## Why use NumPy?
- Faster than Python lists  
- Supports multi-dimensional arrays  
- Provides many mathematical functions  
- Used in Data Science, Machine Learning, and Data Analysis
